## 反模式自检

### 里程碑反模式

- [ ] 里程碑数量 3-5 个(无 < 3 或 > 5)?
- [ ] 里程碑均匀分布(无聚集在一周内)?
- [ ] 每个里程碑有日期+交付物+状态?

### 风险反模式

- [ ] 风险数量 ≥ 3?
- [ ] 风险有空泛(如"可能延期")?——应改为具体风险
- [ ] 风险有责任人(@人)?
- [ ] 风险有应对措施(可操作)?

### 协作反模式

- [ ] 协作有具体负责人(@人),不是"后端/前端"?
- [ ] 协作有时间节点(可追踪)?
